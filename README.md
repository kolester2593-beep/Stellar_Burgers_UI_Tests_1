Автоматизированные тесты для Stellar Burgers

 	Цель проекта
Автоматизация проверки ключевой функциональности веб-приложения заказа бургеров с использованием Selenium WebDriver и pytest.

Покрытие тестами
- Регистрация пользователей (успешная + валидация пароля)
- Аутентификация (4 способа входа в систему)
- Навигация (переходы между разделами)
- Работа конструктора (переключение категорий ингредиентов)
- Выход из системы

Технологический стек
- Язык: Python 3.8+
- Фреймворк: pytest
- WebDriver: Selenium + ChromeDriver
- Менеджер зависимостей: pip
- Паттерн проектирования: Page Object

Статистика
- Всего тестов: 13
- Процент покрытия основной функциональности: ~85%
- Время выполнения: ~60 секунд
- Стабильность: 100% прохождение при корректной среде
