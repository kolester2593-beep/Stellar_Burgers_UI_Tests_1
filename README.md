Автоматизация тестирования Stellar Burgers

## 📌 Описание проекта
Проект содержит автоматизированные UI-тесты для проверки основного функционала веб-приложения [Stellar Burgers](https://stellarburgers.nomoreparties.site/). Тесты написаны на Python с использованием Selenium WebDriver и охватывают ключевые пользовательские сценарии: регистрация, вход в систему, навигация в личном кабинете и работа с конструктором бургеров.

## 🚀 Технологии и инструменты
- **Python 3.x**
- **Selenium WebDriver**
- **Pytest** (фреймворк для запуска тестов)
- Веб-браузеры: **Google Chrome**, **Mozilla Firefox**
